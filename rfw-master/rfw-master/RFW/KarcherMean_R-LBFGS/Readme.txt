"ImportGROPT" - (script) Adds the necessary paths to use the GROPT package
"ClearGROPT" - Removes the necessary paths to use the GROPT package
"Generate_Figure" - Generate the figures in the paper.
"Karcher_mean" - Compute the Karcher mean of a given data set. 

solvers - (directory) Solvers: Algorithms
           It contains Riemannian algorithms. It is independent of problems.
drivers - (directory) Solver drivers for the Karcher mean computation problem.
            The files in it contain objects related to the problem. 
            For example, function, gradient, action of Hessian,retraction, vector transport, etc.
